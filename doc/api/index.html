API Reference

e!Peek

Genome Browser
  • start: Starts the plugin using the defined coordinates
  • right: Pans the genome browser to the right
  • left: Pans the genome browser to the left
  • zoom: Zooms in/out the genome browser
  • allow_drag: Sets the response of the genome browser to mouse panning / zooming
  • height: Gets/Sets the height of the genome browser
  • width: Gets/Sets the width of the genome browser
  • background_color: Gets/Sets the background color of the genome browser
  • foreground_color: Gets/Sets the foreground color of the genome browser
  • ease: Gets/Sets the d3.ease function to use in the genome browser
  • homologues: Retrieves homology information from the REST API
  • species: Gets/Sets the species to be used in the genome browser
  • chr: Gets/Sets the chromosome (or seq_region) to be used in the genome browser
  • from: Gets/Sets the start coordinate to be used in the genome browser
  • to: Gets/Sets the end coordinate to be used in the genome browser
  • gene: Gets/Sets the gene name to be used in the genome browser
  • genes: Gets the genes currently displayed in the genome browser
  • gene_info_callback: Callback called every time a gene is selected
  • gene_search_callback: Callback called every time a gene is searched via the REST api
  • ensgene_search_callback: Callback called every time an Ensembl gene identifier is searched
  • genes_callback: Callback executed after every pan / zoom event has finished

Ensembl REST API
  • call: Makes an asynchronous call to the ensembl REST API
  • url.region: Retrieves the genes included in the specified region
  • url.species_gene: Retrieves the ensembl gene associated with the given name and species
  • url.homologues: Retrieves the homologues (orthologues + paralogues) of the given ensembl ID
  • url.gene: Retrieves the ensembl gene associated with the given ensembl ID
  • url.chr_info: Retrieves the information associated with the given seq_region

Misc Methods
  • scriptPath: Retrieves the path to the file named as its argument
