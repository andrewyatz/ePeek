e!Peek

Drag your mouse, zoom in/out

e!Peek is a JavaScript plug-in that allows the placement of a simple genome browser in any web page or document where custom Javascript is allowed. It is very easy to embed and configure. You only need to insert a few lines of Javascript in your page and let the plug-in do the rest.

#Introduction

A e!Peek plug-in consists of two components. The e!Peek core plug-in and an e!Peek theme that defines how to interact with the core plug-in. Although this may seem very abstract, it is in fact very simple, and converts e!Peek into a very powerful and customizable plug-in.

The easiest way to embed e!Peek in your site is by using one of the distributed themes. The general pattern looks something likes this:

<script src="http://d3js.org/d3.v3.min.js" charset="utf-8"></script>

var gB = epeek().width(920); // other methods can be included here
var gBTheme = epeek_theme(); // other methods can be included here
gBTheme(gB, document.getElementById('DOM_element_id');
