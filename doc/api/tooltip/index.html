Tooltips

#The Tooltip object

The e!Peek plug-in offers its own tooltip library. This has been designed to work well with svg elements, although it also works with other HTML elements. To create a tooltip you need to call the epeek.tooltip method.

Check also how tooltips are used in the epeek.genome plugin

Tipycally, a tooltip is displayed as a consequence of a mouse event (normally, by clicking an element). This is mostly required in order to get the correct x and y coordinates of the element and place the tooltip near the selected element. The following code illustrates how epeek tooltips can be used:


(function() {
   // This is the function that is called on every click on the html element
   // Here we are hard-coding the information that will go into the tooltip
   // But it can also be passed as a parameter.
   var init_tooltip = function() {
      var info = {
                  "header" : { "label" : "My",
                               "value" : "Tooltip"
                             },
                  "rows"   : [ { "label" : "Row1", "value" : "Value1" },
                               { "label" : "Row2", "value" : "Value2" },
                               { "label" : "Row3", "value" : "Value3" }
                             ]
                 };
      epeek.tooltip.call(this).table(info);
   };

   // An svg element is created
   var svg = d3.select("#body").append("svg");
   
   // An svg circle is created inside the svg element
   // and an event-listener is attached to our tooltip-ing function above
   var circle = svg.append("circle")
                   .attr("cx", 50)
                   .attr("cy", 50)
                   .attr("r", 10)
                   .on("click", init_tooltip);
})();

In the code above, epeek.tooltip.call(this) creates a new tooltip on the element referred by this. This tooltip is filled by the tooltip.table method with the information provided as its argument. The call javascript method is needed in order to pass the contextual information to the tooltip and make sure that the coordinates of the clicked element are taken into account when placing the tooltip in the right position

There are two things needed by the epeek.tooltip method to work. The first one is a div element where the tooltip can be attached (outside the svg element). The second one is the coordinates of the clicked element and this is achieved setting the context of the tooltip code with call (see previous paragraph).

# epeek.tooltip()

This method creates a new tooltip on the clicked element. The tooltip itself is attached to the first div ancestor of the element in the DOM tree. This tooltip is empty and needs to be populated with the tooltip.plain, tooltipheader or tooltip.table methods

# tooltip.plain (<string>)

This method populates the tooltip with the string given as its argument. This string can be valid html. For example:


var tooltip_string = '<p>Hello world</p>';
epeek.tooltip.call(this).plain(tooltip_string);

# tooltip.table (<object>)

This method populates the tooltip with a table-based content. When called, the method expects an object with the following fields as its argument:

  • header : {entry_object} -- An object (see below)
  • rows : [{entry_object}, {entry_object},...] -- An array of objects

Where an {entry_object} is an object with the following fields:

  • label : The name of the new field
  • value : Its value

See the code above for an example

# tooltip.background_color ([string])

This method gets/sets the background color of the tooltip. This color should be expressed as a valid hexadecimal number (including the "#" prefix) like "#FFFFFF": 


var tooltip = epeek.tooltip().background_color("#CCDD00");

# tooltip.foreground_color ([string])

This method gets/sets the foreground color of the tooltip. This color should be expressed as a valid hexadecimal number (including the "#" prefix) like "#FFFFFF". This color can be set to the same foreground color as the epeek_genome object:


var tooltip = epeek.tooltip()
                   .background_color("#CCDD00")
                   .foreground_color("#0022FF");
