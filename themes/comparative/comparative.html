The Comparative Theme

Overview

Select different genes and species

In this theme two genome browsers showing the same gene in two different species are displayed. It uses buttons to control the panning and zooming as in the buttons theme. In this case, the buttons are encoded directly in the theme code (not in the html) and buttons to control both genome browsers and each of them independently are used.

This theme is used in the current page as follows:

var gB1 = epeek().species("human").gene(gene).allow_drag(false);
var gB2 = epeek().species("mouse").gene(gene).allow_drag(false);
gBTheme = epeek_theme();
gBTheme([gB1, gB2], document.getElementById("genomeBrowser"));

Check the other themes

If you have checked out other themes, you will probably be familiar with this structure already. In this case because we have more than one instances of the core plug-in to show, this theme accepts an array of core instances as its first parameter instead of a single instance present in other themes. The second parameter is the usual DOM element (div) where the plug-in is inserted.
