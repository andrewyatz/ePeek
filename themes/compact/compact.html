The Compact Theme

Overview

This example shows a more sophisticated theme that includes many others capabilities like searching, jumping to orthologues/paralogues, etc... It also includes links to the Ensembl browser and links to the mobile theme via a QR-code

The structure of the theme is exactly the same as we have seen in the other examples.
