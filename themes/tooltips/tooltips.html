Adding Tooltips

Click on genes to show its information

Overview

By default, the epeek_genome object doesn't react to gene selection (clicking on a gene/gene name). This is so because there is no easy default behavior: each theme may want to react differently to this event. Displaying a tooltip showing basic genome information may look like a reasonable default, so epeek_genome makes this action easy to implement in your theme

Writing a custom tooltip library is not always trivial. The epeek_genome plug-in offers tooltips as an opt-in in your theme. The only thing needed is to plug the epeek.gene_info_callback method of the epeek_genome object to epeek.tooltip in your theme. As in:


var epeek_theme = function() {
    var theme = function(gB, div) {

	gB.gene_info_callback = epeek.genome.tooltip();
	
	gB(div);
	gB.start();
    };

    return theme;
};

Having a different response when clicking on genes is possible by assigning a different callback to gene_info_callback
