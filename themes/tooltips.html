The Tooltip Theme

Click on genes to show its information

Overview

Writing a custom tooltip library is not always trivial. The ePeek core plug-in has tooltips as an opt-in. The only thing needed is to plug the gene_info_callback method of the core plug-in to epeek.tooltip in your theme. As in:


var epeek_theme = function() {
    var theme = function(gB, div) {
	gB.gene_info_callback = epeek.tooltip()
	    .background_color(gB.background_color())
	    .foreground_color(gB.foreground_color());
	
	gB(div);
	gB.startOnOrigin();
    };

    return theme;
};

Having a different response when clicking on genes is possible by assigning a different callback to gene_info_callback
