Overview

Select different genes and species

In the comparative theme two genome browsers showing the same gene in two different species are displayed. It uses buttons to control the panning and zooming as in the buttons theme. In this case, the buttons are encoded directly in the theme code (not in the html) and buttons to control both genome browsers and each of them independently are used.

This theme is used in the current page as follows:

var gBs = [ epeek().species("human").width(width).height(height).gene(gene).allow_drag(false).background_color(bgColor).foreground_color(fgColor),
                    epeek().species("mouse").width(width).height(height).gene(gene).allow_drag(false).background_color(bgColor).foreground_color(fgColor) ];
  gBTheme = epeek_theme();
  gBTheme(gBs, document.getElementById("genomeBrowser"));

Check the other themes

If you have checked out other themes, you will probably be familiar with this structure already, but because we have more than one instances of the core plug-in to show, this theme accepts an array of core instances as its first parameter instead of a single instance present in other themes. The second parameter is the usual DOM element (div) where the plug-in is inserted.

API

This theme has a very simple API with just a couple of methods:

  • theme.species(new_species, position)

    species sets a new species for one of the genome browsers shown. The first argument is the name of the species and the second one is the position of the genome browser (1-based).

  • theme.gene(new_gene)

    gene sets a new gene for all the genome browsers displayed.
